테두리 관련 스타일(border~)

        선택자{
            *테두리 스타일
            border-위치-style : none/dotted/dashed/solid/dobule/groove/ridge/inset/outset;

            *테두리 두께
            border-위치-width : 테두리 두께;

            *테두리 색상
            border-위치-color : 테두리 색상;

            *박스에 그림자효과
            box-shadow: 가로,세로, 흐림, 번짐, 색상;

            *테두리 모서리에 둥글게 하는 스타일
            border-위치-radius : 숫자;

            *테두리 두께,스타일,색상등을 한번에 지정
            border-위치 :두께 스타일 색상;
        }
